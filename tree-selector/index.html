# TreeSelector 树形选择器

演示一个基于 element-ui 二次封装的组件，该组件可以独立使用。

# 安装

npm i @palvue/tree-selector

# 引入

import TreeSelector from '@palvue/tree-selector'
Vue.use(TreeSelector)
